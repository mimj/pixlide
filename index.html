نحوه استفاده

ساختار html برای اسلایدر با عکس های هم اندازه به این صورت است:

            
            <div class="slider-container-same-resolution" id="slider-container">
                <div class="slideWrapper-same-resolution active ">
                    <img src="images/sameres/1.jpg" class="slide">
                </div>
                <div class="slideWrapper-same-resolution">
                    <img src="images/sameres/2.jpg" class="slide">
                </div>
                <div class="slideWrapper-same-resolution">
                    <img src="images/sameres/3.jpg" class="slide">
                </div>
            </div>

ابتدا کلاس اسلایدر را وارد کنید.

            
            var {Slider} =require("../src/Slider.js");

سپس یک شی از کلاس اسلایدر با تنظیمات مورد نظر بسازید و متد Init را فراخوانی کنید.

            
            var s = new Slider({
                sliderId: "slider-container"/* اسلایدر مورد نظر ID*/,
                autoLoopTime: 3000/*زمان مورد نظر برای هر اسلاید*/,
                hasDotNav: false /*true دکمه های پایین اسلایدر برای جابجایی. پیشفرض */,
                nextAndPrevButtons: false, /*true پیشفرض */
            });
            s.init();

اسلایدر با عکس های رزولوشن متفاوت برای مواقعی که عکس ها رزولوشن یکسان ندارند و نمایش تمام عکس نیز اهمیت دارد(مثل آثار هنری)

html برای اسلایدر با عکس های غیر هم اندازه به این صورت است:

            
            <div class="slider-container" id="slider-container2">
                <div class="slideWrapper active ">
                    <img src="images/1.jpg" class="slide">
                </div>
                <div class="slideWrapper">
                    <img src="images/2.jpg" class="slide">
                </div>
                <div class="slideWrapper">
                    <img src="images/3.jpg" class="slide">
                </div>
                <a class="next">❱</a>
                <a class="prev">❰</a>
                <div class="dotsWrapper"></div>
            </div>

چون در این حالت ارتفاع اسلایدر باید ثابت باشد میتوانید برای عرضهای مختلف -به تعداد دلخواه- ارتفاع مناسب اسلایدر را تعیین کنید. به این صورت که (برای مثال) برای عرض های تا 600px ارتفاع اسلایدر 100px و برای عرض حداقل 600px ارتفاع اسلایدر 200px و برای عرض حد اقل 992px ارتفاع اسلایدر 300px و ... .

            
            var s2 = new Slider({
                sliderId: "slider-container2",

                sliderHeights: {
                    "0": "100",
                    "600": "200",
                    "992": "300",
                    "1200": "450",
                },

                autoLoopTime: 4000,
                hasDotNav: true /*true پیشفرض */,
                nextAndPrevButtons: true /*true پیشفرض */
            });
            s2.init();
